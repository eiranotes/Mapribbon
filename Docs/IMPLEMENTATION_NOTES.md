# 구현 메모

## 설계 기준

- 앱 UI는 생산성 도구처럼 중립적으로 유지한다.
- 사진과 지도 결과물만 인쇄물 질감과 리본 연결선을 사용한다.
- 사용자가 여행 중 앱을 실행하거나 위치 추적을 시작하지 않는다.
- 첫 결과 이전에는 회원가입, 템플릿 선택, 결제를 노출하지 않는다.

## Xcode에서 확인할 항목

1. Signing 팀 지정
2. 실제 iPhone의 위치 메타데이터가 포함된 사진으로 테스트
3. 제한된 사진 접근에서 날짜 목록 갱신 확인
4. iCloud 원본 사진의 네트워크 다운로드 확인
5. 한국어/일본어 권한 문구 확인
6. MapKit 스냅샷의 지역별 POI 제거 결과 확인
7. 9:16, 4:5, 3:4 이미지 렌더링 메모리 사용량 확인
8. App Store Connect 비소모성 상품 등록

## MVP 이후 우선순위

1. 대표 사진 자동 선별 품질 개선
2. 위치 없는 사진의 인접 장소 수동 포함
3. 일본 47개 도도부현 Atlas
4. 다일 여행 캐러셀 출력
5. 인쇄용 PDF
