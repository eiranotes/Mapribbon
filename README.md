# MapRibbon iOS MVP

사진의 촬영 날짜와 위치 메타데이터를 이용해 하루 여행을 지도·사진·핀·실로 구성된 공유용 핀보드로 자동 생성하는 SwiftUI 앱입니다.

## 핵심 흐름

```text
위치 사진이 있는 날짜 선택
→ 사진 자동 선택 또는 직접 선택
→ 장소별 자동 묶기
→ 지도 위 사진 핀과 시간순 실 연결
→ 사진·문구·방문 순서 세부 편집
→ Instagram Stories / 사진 저장 / 시스템 공유
```

## 실행

1. `MapRibbon.xcodeproj`를 Xcode에서 엽니다.
2. Signing & Capabilities에서 개발팀을 지정합니다.
3. 실제 iPhone에서 실행합니다. 시뮬레이터는 사진 위치 메타데이터와 Instagram 앱 연동 테스트가 제한적입니다.
4. 영구 구매 테스트를 하려면 App Store Connect 또는 StoreKit Configuration에 `com.eiraworks.mapribbon.lifetime` 비소모성 상품을 등록합니다.

## MVP 포함 범위

- 사진 보관함 권한 및 제한 접근
- 위치 사진 날짜 자동 탐색
- 날짜별 사진 자동 선택 / 직접 선택
- 시간·거리 기반 장소 클러스터링
- 동일 위치 재방문을 별도 정류장으로 보존
- 역지오코딩과 대표 사진 자동 선택
- MapKit 정적 지도 생성
- 지도 위 원형 사진 핀, 사진 수 배지, 시간순 실 연결
- 핀보드 / 리본 / 에디토리얼 / 포스트카드 템플릿
- 위치 사진 핀 터치 시 해당 장소의 사진을 페이지 형태로 펼쳐보기
- Reduce Motion 대응 인터랙티브 사진 갤러리 애니메이션
- 장소명, 방문 순서, 대표 사진, 표시 여부 수정
- 당일 사진을 장소 사이에 추가·제거·이동
- 가장 긴 시간 공백 기준 장소 분리, 이전 장소와 합치기
- 제목과 한 줄 문구 편집
- 9:16 / 4:5 / 3:4 고해상도 렌더링
- Instagram Stories 직접 전달
- 사진 저장 및 시스템 공유 시트
- SwiftData 보드 아카이브
- 한국 17개 지역 Memory Atlas
- StoreKit 2 영구 구매 골격
- 한국어 우선 및 일본어 핵심 문자열

## 편집 정책

- 같은 좌표라도 충분한 시간 간격을 두고 다시 방문하면 별도 정류장으로 유지합니다.
- 같은 장소의 사진을 나누고 싶다면 `사진과 경로` 화면에서 해당 장소를 왼쪽으로 스와이프해 `분리`를 선택합니다.
- 분리는 사진 사이에서 가장 긴 45분 이상의 시간 공백을 기준으로 합니다.
- 잘못 나뉜 장소는 `이전과 합치기`로 되돌릴 수 있습니다.
- 방문 순서는 드래그로 바꾸며, 공유 이미지의 실 연결 순서도 같은 순서를 따릅니다.
- 사진은 장소별 편집 화면에서 다른 장소로 이동하거나 제외할 수 있습니다.

## 의도적으로 제외한 범위

GPS 백그라운드 추적, 실제 도로 경로, 계정, 서버, 광고, 공개 피드, 여행 일정, 영상 생성, 자유형 Canva식 캔버스 편집.

## 디자인 원칙

- 앱 UI는 장식보다 선택·확인·내보내기 속도를 우선합니다.
- 한 화면에는 하나의 주 CTA만 둡니다.
- 앱 화면은 중성적인 생산성 도구로 유지하고, 종이·사진·핀·실의 질감은 결과물에 집중합니다.
- 지도 위 핀은 단순 마커가 아니라 해당 장소의 대표 사진 자체입니다.
- 반복 사용되는 동작은 즉시 반응하고, 사진 스택을 펼치는 드문 동작에만 300ms 미만의 공간 전환 애니메이션을 사용합니다.
- 접근성의 Reduce Motion이 켜져 있으면 이동 애니메이션을 짧은 페이드로 대체합니다.

## 검증 상태

### 기존 `main` 기준

초기 MVP 커밋은 macOS/Xcode에서 iOS 시뮬레이터 빌드와 `PhotoClustererTests` 2건을 통과했습니다.

### `agent/pinboard-photo-pipeline` 기준

현재 변경 작업을 수행한 실행 환경은 Linux이며 `xcodebuild`와 GitHub CLI가 설치되어 있지 않습니다. 따라서 이 브랜치의 신규 SwiftUI 코드와 Instagram 연동은 이 환경에서 직접 컴파일·시뮬레이터 실행하지 못했습니다.

다음 항목은 macOS/Xcode에서 반드시 확인해야 합니다.

1. iOS 17 이상 시뮬레이터 빌드
2. `PhotoClustererTests` 4건 실행
3. 실제 iPhone에서 전체/제한 사진 접근
4. iCloud 원본 다운로드
5. 동일 위치 재방문 표시 및 분리·합치기
6. 사진 핀 터치 갤러리의 빠른 반복 탭과 Reduce Motion
7. 9:16, 4:5, 3:4 이미지 메모리 사용량
8. Instagram 설치/미설치 상태의 Stories 전달
9. 사진 추가 전용 권한과 저장 결과
10. App Store Connect StoreKit 상품
